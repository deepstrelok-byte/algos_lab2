# algos_lab2
Лабораторная работа №2
Тема: Сортировка и поиск
Студент: [Твое Имя], номер в списке: 2

Вариант задания:
1. Метод сортировки: Линейный выбор с подсчетом (Comparison Counting Sort).
2. Структура таблицы: Комбинированный ключ (строка + целое), хранение данных и ключей вместе.

Описание:
Программа реализует ввод таблицы из текстового потока, ее сортировку заданным методом и бинарный поиск по ключу.
Для тестирования используется перенаправление ввода (stdin) из заранее подготовленных файлов.

Формат входных данных:
Данные в файле должны идти построчно в формате:
<строка> <целое_число>

Окончание ввода таблицы обозначается ключевым словом: END_TABLE
Все данные после этого слова считаются ключами, которые необходимо найти в отсортированной таблице.

Список файлов:
- lab2.c : исходный код программы
- test_sorted.txt : тест для упорядоченной таблицы
- test_reverse.txt : тест для таблицы с обратным порядком
- test_random.txt : тест для неупорядоченной таблицы

Компиляция:
gcc lab2.c -o lab2

Запуск тестов:
1. Тест на упорядоченных данных:
   ./lab2 < test_sorted.txt

2. Тест на данных в обратном порядке:
   ./lab2 < test_reverse.txt

3. Тест на случайных данных:
   ./lab2 < test_random.txt

Для сохранения результатов работы в файл можно использовать перенаправление вывода:
./lab2 < test_random.txt > results.txt
